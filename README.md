# Pronto
A cross cantilever 3d printer

# Project goals

- Easily serviceable
- full 24v electrical system
- V6 hotend
- Heated build plate
- Greater than 200x200x200 build volume
- Full linear rails
- 500mm/s or greater max speed
- Direct drive extruder
- Portable
- Belted Z
